AENA - GESTION DE AEROPUERTO

Mejora 1

Dado un aeropuerto, determinar el número de puertas de embarque disponibles.

Mejora 2

Dado un avión, determinar el número de asientos reservados. Este número se corresponde con el número de billetes comprados para un avión determinado.

Mejora 3

Dado un avión, determinar el número de asientos ocupados. Este número se corresponde con el número de personas que están dentro de un avión determinado.

Mejora 4

Listar los aviones con licencias de vuelo caducadas.

Mejora 5

Listar los aviones que utilicen aeropuertos españoles, ya sea porque están en ellos o porque realizan una ruta donde el origen o el destino es español.
